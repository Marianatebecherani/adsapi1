O que é Daily Scrum?

São reuniões realizadas diariamente, dando a cada membro da equipe um tempo para colocar os principais atributos. Aqui informamos a toda a equipe o status da tarefa que foi atribuída ao membro, se está tendo dificuldade ou se há alguma ideia a acrescentar.

Práticas para manter a Daily Scrum eficiente:
  1. O ideal é que ela aconteça logo pela manhã. Além disso, o tempo de duração de uma Daily Scrum deve ser de 15 minutos ou menos.
  2. A Daily Scrum é feita para que os executores do trabalho se auto-organizem. Em geral, isso significa a participação obrigatória dos desenvolvedores.
  3. É interessante definir o local onde a Daily será realizada. Inclusive, é uma prática comum que essas reuniões sejam sempre feitas no mesmo local e horário. Isso ajuda o time a ganhar uma cadência, ter uma disciplina de execução.
  4. Quando surge algum assunto técnico específico de um ou dois membros do time, deve-se informar e tratar desse assunto no pós-daily, ou seja, após a reunião. Para que, assim, não atrase a execução do encontro.
